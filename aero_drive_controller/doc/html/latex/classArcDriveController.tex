\subsection{\-Detailed \-Description}
\-Low level velocity-\/arc based drive controller. 

\-Implements a velocity-\/control based controller which is capable of performing arc based motions. \-There are separate control modes for operation based on the capabilities of the base platform\-: $\ast$\-Without \-Swerve \-Capabilities -\/\-Standard \-Tank \-Steer \-Velocity-\/\-Arc $\ast$\-With \-Swerve \-Capabilities -\/\-Smooth \-Velocity-\/\-Arc -\/\-Translation
